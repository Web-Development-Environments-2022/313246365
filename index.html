Differences between web versions:
Web1 - Non-dynamic, static sites, difficult to update pages, basic design.
The data is presented almost as it stored on the server and accessed to the user by tables for exemple.
The user must know the exact URL to access the wanted information and there were no search engines.
The main use is - read only. Mostly static sites appeared: image pages, articles, forms.


Web2 - The main innovation is that end users can also share content.
Thus the usage was upgraded to read write. Because people started creating the content themselves -
a standard for building websites was created and the design was upgraded.
Moreover, search engines were used and there was no need to manually search for the specific content.
All kinds of collaborative and social sites have been created, such as Facebook and Twitter, news sites and a variety of applications.


Web3 - The age of read write execute. The network tries to present the user with the best information for him,
and adapts itself to the user himself. Inanimate objects can be connected to the network thanks to IoT
and the page design has been upgraded to 3D. Smart applications have been created that combine artificial intelligence. A semantic character was created for
the content and through it connections are created to improve search results.
It should also be noted that during the development of the web, the number of users has increased.


An example of an illustration is the response to the "Side effects of drugs" query in the different webs.
In web1 - we will not be able to get an answer to this query since there was no search engine at all. We must enter
exact URL address. In web2 we can find results on the internet that contain the query search words from various sites.
In web3 we will get exactly the most suitable search results for us.
For example, information from the HMO card in which the records of the medications that the user is taking will be synchronized with the
search and the user will be able to see simulations (or for example three-dimensional holograms) of side effects that occur exactly
from the medications he is taking.
